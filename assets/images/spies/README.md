- spy_1.png
- spy_2.png  
- spy_3.png

You can add more images by updating the `_spyImages` list in `role_reveal_screen.dart`.

Images should be:
- PNG format
- Square aspect ratio recommended  
- Around 200x200px to 500x500px
- Represent spy/impostor characters
